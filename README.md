Print 3 most popular nouns from Habr feed article titles.

![](http://www.userlogos.org/logo/jumpordie/02142011/habrahabrru)

## Requirements

* Python 3.X

## Installation

```
$ make
```

## Usage

```
$ python habr_challenge.py
```

For more information type: ```$ python habr_challenge.py -h```

## Testing
```
$ PYTHONPATH=. pytest
```
